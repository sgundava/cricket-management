# Cricket Management Game

A cricket franchise management simulation game (IPL/Football Manager style).

## Project Structure

```
├── frontend/     # React + TypeScript + Vite
├── backend/      # Coming soon (Node.js API)
├── DESIGN.md     # Game design document
└── CHANGELOG.md  # Version history
```

## Quick Start

```bash
cd frontend
npm install
npm run dev
```

## Deploy to GitHub Pages

```bash
cd frontend
npm run deploy
```
